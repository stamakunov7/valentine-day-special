## Installation

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/stamakunov7/valentine-day-special.git
   ```

2. Navigate to the project directory:
   ```bash
   cd valentine-day-special
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

## Customization

Feel free to customize the content to suit your preferences:

- **Greeting**: Modify the greeting message in `index.html`
- **Song**: Replace the existing song (`song/song.mp3`) with your preferred romantic track
  - *Note: I used "I Wanna Be Yours" by Arctic Monkeys*
- **Images**: Add or replace images in the `img/` directory to personalize the experience
- **Text**: Edit the messages in `index.html` to make it more personal
- **Colors & Styles**: Customize the appearance in `style/style.css`
